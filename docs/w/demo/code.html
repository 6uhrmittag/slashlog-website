Code Demo

Quick reference and showcase for how to use the syntax-highlighter for code.

V1.0.0 last updated 3 months ago on 2018/07/20


This document is intended as a quick reference and showcase. For complete information on highlighter used please see the Highlight.js implementation.

Take a look at the Markdown file to see how these are implemented.

This is some Python code:

def foo(bar):
    pass

This is some JavaScript code:

function look(where) {
    if (where == 'here') {
        alert('woo');
    }
}

We can even highlight markdown code:

# H1
I am **bold** and I am *italic*.

The Atom editor applies the syntax highlighting in the editor!
